# الواجهة الأمامية (Frontend)

هيكل المجلدات جاهز حسب دليل المشروع (`CLAUDE.md`). بناء واجهة React (RTL + TailwindCSS + خط Tajawal)
يبدأ بعد اكتمال الواجهة الخلفية واختبارها بنسبة 100% — وفق منهجية العمل في الجزء الرابع من الدليل.

```
src/
├── assets/       # الصور والخطوط العربية
├── components/   # المكونات المشتركة
├── context/      # إدارة الحالة العامة (Auth, Theme)
├── hooks/        # الخطافات المخصصة
├── pages/        # صفحات النظام
└── services/     # الاتصال بالـ API (Axios)
```
